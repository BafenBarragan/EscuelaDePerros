------------------------------------------------------------------------
Este repositorio contiene los archivos de Código fuente del proyecto de
la base de datos de una escuela de perros en la ciudad de Bogotá, este
ejercicio fue propuesto en el ambiente de aprendizaje como parte 
la guía #3 del bloque de estudios de Construcción del Software.
------------------------------------------------------------------------

- PROJECT TITLE: Colegio de Perros

- PURPOSE OF PROJECT Desarrollar el proyecto de una base de datos para una
escuela de perros.

- VERSION or DATE: 1.0

- HOW TO START THIS PROJECT: El proyecto contiene dos clase, la clase Perro
y la clase EscuelaDePerros, la primera contiene todos los métodos para leer 
y obtener la información de cada perro. La clase EscuelaDePerros permite
inicializar una interfaz de línea de comando para así ejecutar el programa.
